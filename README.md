# User Retention Analysis

## Description
This project was developed as a final task for the Home Credit Indonesia program at Rakamin Academy. It focuses on analyzing user retention patterns to understand customer behavior over time.

The analysis aims to identify trends in user activity, retention rates, and key factors influencing user engagement, providing insights to support data-driven decision making in the financial services domain.

## Dataset
The dataset consists of customer and behavioral data provided within the Home Credit project scope, used to analyze retention and engagement patterns.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
